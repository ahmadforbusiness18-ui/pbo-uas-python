## Deskripsi
Projek ini merupakan Final Project mata kuliah. Pemrograman Berorientasi Objek (PBO), terkait Sistem Informasi Akademik

Aplikasi ini bertujuan untuk mengelola data mahasiswa dan nilai, meliputi:

- Menambahkan data mahasiswa

- Menampilkan data mahasiswa

- Mengubah data nilai

- Menghapus data mahasiswa

- Menghitung dan menampilkan nilai akhir

Program ini menerapkan konsep Object-Oriented Programming (OOP) seperti class, object, encapsulation, dan method, sehingga struktur kode menjadi lebih rapi dan mudah dipahami.

---
## Cara menjalankan Program
1. Buka program main
2. pilih menu 1 untuk tambah data mahasiswa (NIM dan Nama)
3. Pilih menu 2 untuk tambah data mata kuliah (Kode MK dan nama MK)
4. PIlih menu 3 untuk input nilai (pilih sesuai kode mulai dari 0-seterusnya)
5. Input nilai mulai dari nilai kehadiran, nilai tugas, nilai uts, dan nilai uas. Dan kemudian muncul nilai rata-rata dari keseluruhan nilai
6. Pilih menu 4 untuk tampilkan data yang telah di input maka akan muncul (Nama mahasiswa | Nama mata kuliah | Nilai)
7. Pilih Menu 5 untuk keluar dari program.
